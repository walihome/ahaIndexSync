# AI 内容打分标准

用于指导大模型对每条内容打 aha_index 分数（最终归一化到 0.0-1.0）。

## 第一步：基础分（选一个，互斥）

| 内容类型 | 基础分 |
|---|---|
| 重大发布、突破性进展、行业里程碑 | 3 |
| 新工具、新产品、新开源项目、新学习资源 | 2 |
| 行业动态（收购、裁员、融资、合作） | 2 |
| 普通资讯、教程、观点文章 | 1 |

## 第二步：加分项（可叠加，每项 +1）

| 维度 | 判断标准 |
|---|---|
| 娱乐性 | 八卦、大佬撕逼、宕机事故、反转剧情，读者喜闻乐见 |
| 福利性 | 免费资源、限免 App、薅羊毛机会、Pro 福利发放 |
| 时效性 | 今天刚发生，有强烈的"现在就要看"紧迫感 |
| 实用性 | 读者看完能立刻用上，解决实际问题 |
| 稀缺性 | 顶级人才动向、冷门但高价值的信息、独家披露 |

## 第三步：减分项（可叠加，每项 -1）

| 维度 | 判断标准 |
|---|---|
| 同质化 | 与近期已有内容高度重复，信息增量低 |
| 营销味 | 明显是软文、自我宣传、付费推广 |
| 信息量低 | 标题党，正文空洞，读完没有收获 |

## 归一化公式

```
最低原始分 = 0（基础分1 + 无加分 - 2个减分）
最高原始分 = 8（基础分3 + 5个加分项）
aha_index = clamp(原始分 / 8, 0.0, 1.0)
```

## 打分示例

| 内容 | 基础分 | 加分 | 减分 | 原始分 | aha_index |
|---|---|---|---|---|---|
| GPT-5 正式发布 | 3 | 时效+1 稀缺+1 | - | 5 | 0.63 |
| ChatGPT Pro 六月免费领 | 2 | 福利+1 时效+1 | - | 4 | 0.50 |
| OpenAI 大规模裁员 | 2 | 娱乐+1 时效+1 | - | 4 | 0.50 |
| 某 AI 工具使用教程 | 1 | 实用+1 | 营销-1 | 1 | 0.13 |
| 某公司又发了篇博客 | 1 | - | 信息量低-1 | 0 | 0.00 |
